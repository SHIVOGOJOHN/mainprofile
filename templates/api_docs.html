API Documentation

Prediction API

This API allows you to get predictions from the federated learning model for Store A.

Endpoint

POST https://federatedsys.onrender.com/fed_sys

Request Body

The request body should be a JSON object with the following fields:

  • Age: integer
  • Income: float
  • Loyalty_Score: integer
  • Prior_Purchases: integer
  • Avg_Spend: float
  • Recency: integer
  • Browsing_Time: float
  • Clicks_On_Promo: integer
  • Purchase_Frequency: float
  • High_Value_Customer: integer
  • Engagement_Score: integer
  • Gender_Male: integer
  • Promotion_Type_Discount: integer
  • Promotion_Type_FlashSale: integer
  • Promotion_Type_LoyaltyPoints: integer
  • Channel_Email: integer
  • Channel_In_store: integer
  • Channel_SMS: integer
  • Time_of_Day_Evening: integer
  • Time_of_Day_Morning: integer

Example Request


{
  "Age": 35,
  "Income": 50000.0,
  "Loyalty_Score": 500,
  "Prior_Purchases": 10,
  "Avg_Spend": 75.5,
  "Recency": 30,
  "Browsing_Time": 120.0,
  "Clicks_On_Promo": 2,
  "Purchase_Frequency": 0.5,
  "High_Value_Customer": 0,
  "Engagement_Score": 3,
  "Gender_Male": 1,
  "Promotion_Type_Discount": 0,
  "Promotion_Type_FlashSale": 1,
  "Promotion_Type_LoyaltyPoints": 0,
  "Channel_Email": 1,
  "Channel_In_store": 0,
  "Channel_SMS": 0,
  "Time_of_Day_Evening": 1,
  "Time_of_Day_Morning": 0
}

Response

The response will be a JSON object with the following fields:

  • prediction: The binary prediction (0 or 1).
  • probability: The prediction probability (float).
  • model_version: The version of the model used for the prediction.
  • last_updated: The timestamp of when the model was last updated.
  • feature_importances: A JSON object with the SHAP values for each feature.
  • shap_plot_data: A JSON object with the data needed to generate a SHAP force plot.
